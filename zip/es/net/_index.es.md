<p>Aspose.ZIP para .NET es una API de manipulación de archivos y compresión de documentos flexible para el formato ZIP estándar. La API permite que las aplicaciones .NET logren la compresión/descompresión de archivos, el archivado de archivos y carpetas y el cifrado de archivos. Esta API funciona con todos los formatos de archivo conocidos: ZIP, RAR, 7Zip, GZIP, BZ2. Y con muchos formatos de archivo de Linux: CPIO, TAR, Lzip, Bzip2, XZ, Z. Brinda protección a través de contraseñas definidas por el usuario y encriptación tradicional usando ZipCrypto o encriptación AES como AES128, 192 y AES256.</p>
