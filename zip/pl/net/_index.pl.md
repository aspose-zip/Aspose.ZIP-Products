<p>Aspose.ZIP dla .NET to elastyczny interfejs API do kompresji dokumentów i manipulacji archiwami dla standardowego formatu ZIP. API umożliwia aplikacjom .NET kompresję/dekompresję plików, archiwizację plików oraz folderów i szyfrowanie archiwów. Ten interfejs API działa ze wszystkimi dobrze znanymi formatami archiwów: ZIP, RAR, 7Zip, GZIP, BZ2. I z wieloma formatami archiwów z systemu Linux: CPIO, TAR, Lzip, Bzip2, XZ, Z. Zapewnia ochronę za pomocą haseł zdefiniowanych przez użytkownika i tradycyjnego szyfrowania przy użyciu szyfrowania ZipCrypto lub AES, takiego jak AES128, 192 i AES256.</p>
