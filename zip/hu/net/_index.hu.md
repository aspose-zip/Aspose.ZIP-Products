<p>Az Aspose.ZIP for .NET egy rugalmas dokumentumtömörítési és archívumkezelési API szabványos ZIP formátumhoz. Az API lehetővé teszi a .NET-alkalmazások számára fájlok tömörítését/kitömörítését, fájlok és mappák archiválását és az archívumok titkosítását. Ez az API az összes jól ismert archív formátummal működik: ZIP, RAR, 7Zip, GZIP, BZ2. És számos Linux archív formátummal: CPIO, TAR, Lzip, Bzip2, XZ, Z. Védelmet nyújt a felhasználó által definiált jelszavakkal és a hagyományos titkosítással, ZipCrypto vagy AES titkosítással, például AES128, 192 és AES256.</p>
